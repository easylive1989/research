# Research 研究專案

這個專案用於執行和記錄各種研究，每個研究都會在獨立的目錄中進行。

## 目錄結構

每個研究會在專案根目錄下建立一個獨立的目錄，命名格式為：

```
YYYY-MM-DD-研究主題名稱/
```

例如：
- `2025-11-08-machine-learning-experiment/`
- `2025-11-15-data-analysis/`

## 研究目錄內容

每個研究目錄必須包含：
- **README.md**：說明研究目的、方法、結果等

其他內容依研究性質而定，可能包含：
- 程式碼檔案
- 資料檔案
- 圖表或視覺化結果
- 研究筆記或文件
- `.gitignore`（依需求自行設定）

## 使用方式

1. 開始新研究時，建立新的研究目錄
2. 在該目錄中建立 README.md 說明研究內容
3. 在該目錄中進行研究相關的開發和記錄
4. 完成後，研究結果和程式碼都保存在該目錄中
